## 版本說明

### 版本號格式
- **主版本號**：重大功能更新或架構變更
- **次版本號**：新功能增加
- **修訂版本號**：Bug 修復和效能優化

### 版本類型
- **Alpha**：內部測試版本
- **Beta**：公開測試版本
- **RC**：候選發布版本
- **Stable**：穩定發布版本
